# Live your life, live your flight ✈️

Proyecto Week-03 - JavaScript Moderno Bootcamp

## Dominio
App de planificacion de viajes.

## Objetivo
Implementar un sistema de gestión completo utilizando Programación Orientada a Objetos con sintaxis ES2023.

## Arquitectura

BaseItem (abstracta)
- Flight
- Hotel
- Activity

Person
- Traveler
- Admin

Sistema Principal
- TravelSystem
- Booking
- History

## Características
- Clases ES2023
- Campos privados (#)
- Herencia y polimorfismo
- CRUD con validación
- Estados activo/inactivo
- Búsqueda y filtrado
- Static blocks
- Integración funcional con DOM
